# 2025 VT GIS Year in Review
This repository contains the code used to serve the annual GIS year in review. It is an ESRI storymaps application wrapped in an html container to serve the official state of vermont header.

The PRD index.html is served in the $web/review/2025/ directory from Azure. Thus it is loaded from the following URL:
https://map.vermont.gov/review/2025

(https://map.vermont.gov/review/2025/index.html)
