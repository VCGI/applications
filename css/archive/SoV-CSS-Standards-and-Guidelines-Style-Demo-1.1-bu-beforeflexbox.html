Media and Content

Images are responsive and will not exceed the width of their container.

Diagrams should be mostly grayscale and use varying line weights and types as well as grayscale fills (where applicable) to convey hierarchy. SVG's with embedded or flattened/rendered fonts are preferred.

HTML-Specific Components

The following components require specific HTML markup with the correct CSS classes to be styled properly.

Table of Contents

The Table of Contents is automatically generated. Any `h2` heading with an `id` attribute will be added as a top-level item.

<h2 id="my-section">My Section Title</h2>

My Section Title

Heading Anchors

<h2 id="section">Section Title
  <a href="#section" class="heading-permalink">
    <i class="fas fa-link"></i>
  </a>
</h2>

Section Title

Alerts & Notices

<div class="alert alert-info">
  <p class="alert-title">Note</p>
  <p>This is an informational alert.</p>
</div>

Note

This is an informational alert.

Task Lists

<ul class="task-list">
  <li>
    <input type="checkbox" id="ex-task1">
    <label for="ex-task1">A task to do.</label>
  </li>
  <li>
    <input type="checkbox" id="ex-task2" checked>
    <label for="ex-task2">A completed task.</label>
  </li>
</ul>
